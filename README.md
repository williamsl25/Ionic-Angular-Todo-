### 1) install Cordova
```
npm install -g cordova
```
### 2) Install Ionic
```
npm install -g ionic
```
### 3) Create the project
```
ionic start todo blank
```
```
cd todo && ls
```
### 4) Configure Platforms
```
ionic platform add ios
```
```
ionic platform add android
```
### 5) Test it
```
ionic build ios
```
```
ionic emulate ios
```
